#  E-commerce Data Analysis & Statistical Testing

## Описание проекта
Комплексный анализ данных электронной коммерции с целью выявления скрытых закономерностей в поведении пользователей и проверки продуктовых гипотез. Проект включает в себя полный цикл работы с данными: от очистки и EDA до проведения статистических тестов и оформления строгой технической документации.

##  Бизнес-задачи
* Анализ ключевых продуктовых метрик и выявление узких мест в воронке продаж.
* Проверка гипотез о поведении пользователей с использованием статистического аппарата.
* Оценка влияния [указать параметр, например: различных типов подписки/сезонности/интерфейса] на средний чек.

##  Стек технологий
* **Язык:** Python
* **Библиотеки:** Pandas, NumPy (обработка данных), SciPy (статистика), Matplotlib, Seaborn (визуализация)
* **Документация:** LaTeX (строгая верстка математических обоснований и выводов)
* **Методологии:** A/B-тестирование, EDA, ETL

##  Ключевые этапы и методология
1. **Предобработка данных (ETL):** Очистка сырых данных, обработка пропущенных значений и аномалий, приведение типов.
2. **Разведочный анализ (EDA):** Построение распределений, выявление корреляций, сегментация пользователей.
3. **Проверка статистических гипотез:** 
   * Использован **T-критерий Уэлча (Welch's t-test)** для сравнения средних значений в независимых выборках с неравными дисперсиями.
