#  Predictive Analytics & Statistical Testing for E-Commerce

## Описание проекта
Комплексный анализ данных электронной коммерции для оптимизации маркетингового бюджета[cite: 2]. Проект включает в себя проверку статистической разницы в среднем чеке (AOV) между домашними (UK) и международными клиентами, а также построение ML-модели для предсказания удержания (Retention)[cite: 2]. 

##  Бизнес-задачи
* Оценка влияния географического положения на средний чек пользователя[cite: 2].
* Построение предиктивной модели машинного обучения на основе исторического поведения[cite: 2].
* Разработка рекомендаций по перераспределению бюджета и удержанию клиентов из зоны риска оттока[cite: 2].

##  Стек технологий
* **Язык:** Python
* **Machine Learning & Статистика:** scikit-learn (Random Forest Classifier, ROC-AUC), SciPy (Welch's t-test)[cite: 2].
* **Обработка данных:** Pandas, NumPy[cite: 2].
* **Визуализация:** Matplotlib, Seaborn (ROC-кривые, KDE-графики)[cite: 2].

##  Ключевые этапы и методология
1. **Предобработка данных (ETL):** Очистка сырого датасета от транзакций без Customer ID и фильтрация возвратов (отрицательные значения)[cite: 2].
2. **Статистическое тестирование:**
   * Сформированы независимые массивы данных и применен T-критерий Уэлча (Welch's Two-Sample t-test)[cite: 2].
   * Доказано статистически значимое различие чеков с p-value < 0.001[cite: 2].
3. **Предиктивная аналитика (ML):**
   * Проведен базовый RFM-анализ (Recency, Frequency, Monetary) для создания признаков модели[cite: 2].
   * Для предотвращения утечки данных (data leakage) в качестве предикторов использованы только Recency и AOV[cite: 2].
   * Обучена модель Случайного леса (Random Forest), достигнут показатель точности классификации AUC-ROC = 0.79[cite: 2].

##  Главные бизнес-выводы
* Международные клиенты приносят статистически больший доход, что визуально подтверждается распределением с длинным «хвостом» на KDE-графике[cite: 2]. Рекомендовано направить часть маркетингового бюджета на глобальные кампании привлечения[cite: 2].
* Исторические данные позволяют надежно прогнозировать возвращаемость клиентов[cite: 2]. Рекомендовано внедрить разработанную ML-модель для автоматической рассылки таргетированных retention-скидок[cite: 2].
